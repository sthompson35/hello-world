Overview

This repository represents a personal milestone in my journey as an African American developer stepping into the world of software engineering. Originally created in 2020 as a foundational hello-world project, it has since evolved into a space dedicated to learning, exploration, and growth in repository-based programming.

Goals

Understand and explain all operating functions of this program
Learn the fundamentals of version control with Git and GitHub
Build a strong foundation in software development
Document my learning process for myself and others
Cultivate a habit of continuous learning through hands-on experimentation
Practice structuring and organizing code in a maintainable way

What This Repository Is About

This repository serves as both a learning environment and a personal portfolio in progress. It is designed to:
Document the learning process transparently for personal reflection and to assist others on a similar path
Gain hands-on experience with repository management, including commits, branches, pull requests, and merges
Practice writing and organizing code
Explore how code runs and functions
